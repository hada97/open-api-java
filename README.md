## 🚀 Como Começar

Para começar a utilizar este projeto, siga as etapas abaixo:

1. Clone o repositório:
    ```bash
    git clone https://github.com/hada97/
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd 
    ```
3. Instale as dependências:
    ```bash
    npm install
    ```
4. Inicie o servidor:
    ```bash
    npm start
    ```
